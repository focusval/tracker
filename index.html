Режим редагування

Введи fine-grained token GitHub із правом Contents: Read and write для цього репозиторію (як створити — див. README). Токен зберігається лише у цьому браузері.

Додавай .ply (uncompressed із SuperSplat) або .splat. Файл конвертується у .splat і комітиться у scenes/. Понад 95 МБ GitHub не прийме. «Експорт карти» збирає весь сайт зі сплатами в один ZIP.

Тапни групу нижче — відкриються тільки її повзунки, щоб карту було видно.

Позиція — перетягни жовтий маркер на карті.

Вирівнює нахили й садить сцену на землю. Найкраще — після ввімкнення обрізки.

Найточніший спосіб. Постав 3+ пари: точка на сплаті → те саме місце на карті. Кнопка «Накласти» підбере масштаб, поворот і позицію.

Пар точок: 0 (треба ≥3)

Межі тягни на карті за ручки ↔ ↕, поворот — за ручку ⟳. Небо зрізай «Висота до», туман — «Висота від».

Увімкни гумку і малюй по зайвих деталях. За замовчуванням стирає лише ближній бік (напр. один фасад). Карта не рухається; «↶» скасовує мазок.

Записує обрізку і стирання у файл. Оригінал → *.orig.splat.
